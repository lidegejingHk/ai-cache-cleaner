# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2025-01-14

### Added
- Activity Bar icon for one-click access from sidebar
- Welcome panel with "Open Dashboard" button
- Better category classification (Visualization, Other)

### Changed
- Improved marketplace description

## [0.1.0] - 2025-01-13

### Added
- Initial release
- Auto-detect AI tools (Claude, Gemini, Cursor, Copilot, etc.)
- Dynamic search for AI tool directories
- Safety level system with scientific definitions
- Customizable safety levels with persistence
- Dark mode OLED theme
- Size visualization and summary cards
- Confirmation dialogs for risky operations
- Reset to default safety levels

### Supported Tools
- Claude Code
- Gemini/Antigravity
- Cursor
- GitHub Copilot
- Codeium
- Continue
- Tabnine
- Amazon CodeWhisperer
- Sourcegraph Cody
- Windsurf
